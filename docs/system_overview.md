# System Overview

This portfolio repository is organized as a small, production-style software system.

## Goals
- Present a developer profile in executable and readable formats.
- Demonstrate engineering fundamentals: architecture, testing, observability, and documentation.

## Main Modules
- `sujal.py`: executable resume profile.
- `scripts/metrics_dashboard.py`: performance and reliability metrics utility.
- `components/`: reusable profile metadata artifacts.
- `tests/`: verification suite across quality categories.
